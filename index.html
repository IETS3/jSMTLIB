jSMTLIB Project

The jSMTLIB project develops Java libraries and tools to support the SMT-LIB standard. This page lists the resources available.

  • A tutorial about the SMT-LIB language.
  • A software application and library for using SMT-LIB (past versions are here).
  • A User Guide for the application, library, and plug-in.
  • An Eclipse plug-in site that is a wrapper for the jSMTLIB library, creating a GUI tool for using SMT-LIB and SMT solvers.

jSMTLIB is hosted at github:

  • The github project is at https://github.com/smtlib/jSMTLIB
  • The github project page (this page) is at http://smtlib.github.io/jSMTLIB/.
  • The github wiki is at https://github.com/smtlib/jSMTLIB/wiki.

Support for some of the development of these resources was provided by GrammaTech, Inc., a user of SMT tools and a creator and vendor of program analysis research and tools.
